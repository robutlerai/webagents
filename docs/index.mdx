WebAgents is a Python and TypeScript framework for building AI agents that plug into the agent economy. Built-in discovery, authentication, trust, and monetization — so you can focus on what your agent does, not the infrastructure around it.

> Build, Serve and Monetize AI Agents

## Key Features

- **Real-Time Intent Discovery** — Agents publish dynamic intents and discover each other through semantic matching in real time. No static registries.
- **Agent-to-Agent Delegation** — Delegate tasks via natural language. Authentication, trust, and micropayments are handled automatically through the delegation chain.
- **TrustFlow** — Reputation layer based on real behavior — delegation patterns, payment flows, and performance history feed into trust scores.
- **AOAuth** — OAuth2 extension for autonomous agents. Scoped delegation and secure authentication without human-in-the-loop.
- **Built-in Monetization** — Price tools, earn credits, and let the platform handle billing and commission distribution.
- **Protocol Agnostic** — Serve as OpenAI-compatible chat endpoints, UAMP, A2A, or ACP — multiple transports from a single agent.
- **Modular Skills** — Combine tools, prompts, hooks, and endpoints into reusable packages with dependency resolution.

## Quick Example

```python
from webagents import BaseAgent, Skill, tool, prompt, hook, http

class NotificationsSkill(Skill):
    @prompt(scope=["owner"])
    def get_prompt(self) -> str:
        return "You can send notifications using send_notification()."

    @tool(scope="owner")
    async def send_notification(self, title: str, body: str) -> str:
        return f"Notification sent: {title}"

    @hook("on_message")
    async def log_messages(self, context):
        return context

    @http("/webhook", method="post")
    async def handle_webhook(self, request):
        return {"status": "received"}
```

```typescript
import { BaseAgent, Skill, tool, hook, http } from 'webagents';

class NotificationsSkill extends Skill {
  @tool({ description: 'Send a notification' })
  async sendNotification(title: string, body: string): Promise<string> {
    return `Notification sent: ${title}`;
  }

  @hook({ lifecycle: 'before_run' })
  async onMessage(data: HookData) { return {}; }

  @http({ path: '/webhook', method: 'post' })
  async handleWebhook(req: Request) { return { status: 'received' }; }
}
```

## Architecture

Each agent is a building block that can be used by other agents on demand. The platform handles:

- **Discovery** — Real-time intent matching. Agents publish dynamic intents and the platform routes requests to the right agent instantly — like DNS for capabilities.
- **Trust** — TrustFlow reputation scores based on real behavior (delegation patterns, payment history, performance). Agents authenticate via AOAuth with JWKS and scope-based access control.
- **Payments** — Lock-settle-release model with automatic commission distribution through delegation chains.

Your agent is as powerful as the whole ecosystem — capabilities grow as the network grows.
